PYR简介

BDSpyrunner(以下简称pyr)是一个运行在BDS(Bedrock Dedicated Server)的的插件加载平台

通过使用CPython与Detours给Python提供多种监听器和API函数，有效简化了插件的开发成本

安装PYR

1.下载BDXCore并解压到BDS根目录
2.运行Rodb.exe
3.然后运行BDS 确保BDXCore成功加载
4.将pyr放入bdxcore_mod文件夹
5.前往Python官网 下载安装Python3.7.9 记得勾选添加到环境变量
6.再次运行BDS，如果看到[BDSpyrunner] x.x.x loaded.，则pyr加载成功了
7.将插件放入./plugins/py，再次运行BDS，就可以正常使用了

插件开发

Pyr成功加载后，即可开始编写插件了，可在以下页面获得一些帮助。

2.API接口-mc

1.mc

解释:mc模块的函数

1.getVersion() -> int

>> 获取pyr代码版本

代码示例 
PyrVersion = mc.getVersion()
print(PyrVersion)
运行结果 
200

2.logout(输出文本:str) -> none

>> 向控制台发送输出消息(可拦截)

代码示例 
mc.logout('我向控制台输出了114514\n')

3.runcmd(指令:str) -> none

>> 执行一条控制台命令

代码示例 
mc.runcmd('give twoone3 apple')

4.setListener(监听器:str,函数:function) -> none

>> 设置function作为str事件监听器，function必须是只有一个参数的函数，事件触发时function将被调用，这个参数是附带的数据

代码示例 
def onPlaceBlock(data):
    print(f'操作方块所在位置:{data["position"]}')
    print(f'方块ID:{data["blockid"]}')
    print(f'方块名称:{data["blockname"]}')
    print(f'玩家Entity对象:{data["player"]}')
mc.setListener('放置方块',onPlaceBlock)
运行结果 
操作方块所在位置:[-69, 3, -50]
方块ID:2
方块名称:minecraft:grass
玩家Entity对象:msz728

5.setCommandDescription(指令:str,描述:str) -> none

>> 设置指令说明，a1为指令字符串，a2为描述，方便玩家补全

代码示例 
mc.setCommandDescription('cmd','我注释了/cmd指令')

6.setServerMotd(Motd:str) -> none

>> 设置服务器外显名字

代码示例 
mc.setServerMotd('我设置了Motd')

7.getBlock(x坐标:int,y坐标:int,z坐标:int,维度:int) -> dict

>> 获取指定位置方块信息

代码示例 
block = mc.getBlock(1,2,1,1)
print(block)
运行结果 
{'blockname':'minecraft:grass','blockid':2}

8.setBlock(方块:str,x坐标:int,y坐标:int,z坐标:int,维度:int) -> none

>> 设置指定位置的方块，a1是方块名称，大小写必须正确，否则会失败

代码示例 
mc.setBlock('minecraft:grass',1,1,1,1)

9.getStructure(x坐标1:int,y坐标1:int,z坐标1:int,x坐标2:str,y坐标2:int,z坐标2:int,维度:int) -> str/none

>> 获取两个坐标之间的结构，did是维度ID，返回值是结构JSON字符串

代码示例 
Structure = mc.getStructure(0,0,0,1,1,1,1)
print('返回类型:',type(Structure))
运行结果 
返回类型: str

10.setStructure(结构JSON:str,x坐标:int,y坐标:int,z坐标:str,维度:int) -> none

>> 在xyz设放置一个结构，did是维度ID，struct是结构JSON字符串

代码示例 
mc.setStructure('{...}',0,0,0,1)

11.getPlayerByXuid(Xuid:str) -> Entity对象

>> 根据Xuid获取玩家

代码示例 
Player = mc.getPlayerByXuid('1145141919810')
print('玩家名:',Player.name)
运行结果 
玩家名: abcd

12.getPlayerList() -> list

>> 获取玩家列表

代码示例 
Player_List = mc.getPlayerList()
for i in Player_List:
    print('玩家名:',i.name)
运行结果 
玩家名: abcd
玩家名: msz728
玩家名: twoone3

13.setDamage(伤害值:int) -> none

>> 设置生物受伤的伤害值

#注:此函数仅在onMobHurt事件下调用才能生效

代码示例 
def onMobHurt(e):
    mc.setDamage(5)
mc.setListener('onMobHurt',onMobHurt)

14.explode(x坐标:float,y坐标:float,z坐标:float,维度:int,强度:float,是否破坏方块:bool,范围:float,是否着火:bool) -> none

>> 产生一个爆炸

代码示例 
mc.explode(1.0,1.0,1.0,1,1.1,True,3.1,False)
